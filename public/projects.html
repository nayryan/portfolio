Emergency waitlist

I developed a hospital waiting list application, implementing an interactive user interface and server-side data management. Here's a brief overview of my accomplishments: User Interface Design: I used HTML and CSS to structure and style the application, making the interaction with forms and waiting lists visually intuitive. Client-side Interactivity: I incorporated JavaScript to enhance the user experience with real-time form validations and dynamic updates to the interface, without needing page reloads. Server-side Processing and Data Management: With PHP, I developed server logic to process user inputs, communicate with the database, and respond to front-end requests. Database and Waiting List Management: I used SQL to create and query the database, storing patient information and efficiently managing their position in the waiting list.
